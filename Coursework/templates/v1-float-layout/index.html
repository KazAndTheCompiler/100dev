Understanding Floats

This is the main content area. In classic layouts, we use floats to create column-based designs.

When we apply float: left to an image, it removes the image from normal document flow and allows text to wrap around it. This is how we traditionally created magazine-style layouts.

The float property takes the element out of normal document flow and positions it to the left or right of its containing block. Other inline elements (like text) will then wrap around the floated element.

Remember: floated elements are taken out of normal document flow, which means they don't affect the height of their container. This is why we need clearfix techniques to contain floats properly.

Without proper containment, parent containers can collapse, leading to layout issues. The clearfix hack solves this by forcing the container to expand and contain its floated children.
