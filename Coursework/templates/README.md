# Template Library

This directory contains centralized, versioned templates for the 100Devs coursework to avoid duplication and ensure consistency.

## Available Templates

### v1-basic
Basic HTML/CSS/JS template with modern standards:
- HTML5 semantic structure
- Mobile-responsive meta tags
- Normalize.css for cross-browser consistency
- Box model and clearfix utilities
- Standard JavaScript placeholder

## Usage

To use a template in a class project:
1. Copy the template directory to your project location
2. Customize the content while maintaining the basic structure
3. Update the meta tags and content as needed

## Benefits

- **Consistency**: All students start with the same well-structured template
- **Maintenance**: Update templates once and all projects benefit
- **Version Control**: Track template improvements over time
- **Reduced Duplication**: Eliminates scattered template copies across class directories
