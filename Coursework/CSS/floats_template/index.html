Image with Text Wrap

Demonstrates how text wraps around floated images — the original purpose floats were designed for.

Article Title

This text wraps around the floated image. The image has float: left applied, removing it from normal document flow and positioning it to the left of its container.

This is the use case CSS floats were originally designed for — magazine-style text-wrap around images. Multi-column page layouts came later as developers found creative (mis)uses for the property.

Notice how margin controls the space between the image and text. Without margin, the text would sit flush against the image edge.
