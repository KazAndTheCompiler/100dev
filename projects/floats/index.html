Image with Text Wrap

Demonstrates how text wraps around floated images

Article Title

This text wraps around the floated image. The image is removed from normal document flow and positioned to the left, allowing text to flow around it. This technique was commonly used for magazine-style layouts before modern CSS layout methods became available.

Notice how the text naturally wraps around the image. You can control the amount of space around the image using margin properties.
